科研（项目）经历

基于强化学习的医学影像配准方法研究   项目负责人   2024.09-2026.04

医学影像配准是临床手术导航与放疗靶区规划的核心前置技术。传统的迭代优化算法（如SimpleITK、Elastix）在面对大幅度初始空间错位时，极易陷入局部极小值导致配准发散，且单次计算耗时通常达秒级甚至分钟级，难以满足术中实时的严苛临床需求。本项目针对上述痛点，创新性地将二维单模态刚性配准任务建模为连续空间马尔可夫决策过程（MDP）。通过深度强化学习范式赋予智能体自主纠偏与全局寻优能力，有效解决了复杂解剖结构下大规模形变对齐的难题。

  • 个人职责：作为项目核心开发者，我独立主导了该配准框架的端到端环境搭建与所有核心实验落地。在研究过程中，我着重针对医学影像场景的特殊性进行算法适配，深入分析了解剖结构的几何拓扑属性，并将物理先验知识显式地融入智能体决策逻辑中。同时，针对强化学习训练初期的策略发散与状态部分可观测（POMDP）等底层难题，独立设计并执行了多阶段渐进式训练实验与网络架构调整，确保了算法在医疗环境下的绝对可靠性与收敛性。
  • 技术路线：本项目构建了融合物理先验的“几何-语义”双流决策网络（Actor），通过最优传输（OT）理论提取解剖结构质心作为平移先验，并将旋转与缩放等高阶形变交由语义网络预测残差，大幅收窄了高维探索空间；同时，针对状态部分可观测难题，设计了基于 Transformer 自注意力与因果掩码机制的时序价值评估网络（Critic），通过深度编码历史行为轨迹精准捕获长程因果一致性，严厉惩罚了破坏全局拓扑的投机性动作；为克服强化学习的冷启动与价值高估陷阱，实施了包含“自监督预训练、价值评估网络热身与融合行为克隆（BC）正则化的联合微调”的三阶段课程学习机制，结合目标策略平滑与 OU 惯性噪声，最终保障了复杂医疗场景下策略的稳定收敛。
  • 项目成效：在单模态二维CT数据集上的严格基准测试表明，本框架成功跨越了0.5秒的实时性门槛，将单次配准平均推理时间极速压缩至0.3799秒，在执行效率上对Elastix（4.9741秒）等传统算法实现了数量级的超越。同时，在应对大幅度随机初始空间错位与像素噪声干扰时，本模型的互信息（MI=0.8101）与归一化互相关（NCC=0.9602）对齐精度均高度媲美临床金标准软件。该成果证实了强化学习范式在解决医疗图像复杂配准任务中的巨大落地潜力。

基于强化学习与博弈论的骗子酒馆策略研究   项目负责人   2025.11-2026.01

非完全信息博弈中的策略寻优一直是人工智能领域的核心挑战。在《骗子酒馆》（Liar’s Bar）这类带有“俄罗斯轮盘赌”不可逆淘汰机制的生存博弈中，传统零和博弈模型难以平衡欺骗收益与生存概率。本项目针对该环境的高随机性与非平稳特征，将游戏严密建模为部分可观测随机博弈（POSG），并提出了一种融合 Transformer 与近端策略优化（PPO）的深度强化学习框架，旨在复杂多方混战中演化出具备动态贝叶斯风险管理能力的理性智能体。

  • 个人职责：作为项目独立开发者，我主导了从博弈数学建模到多智能体强化学习架构落地的全流程。秉持着“算法需深度适配特定场景”的研究理念，我并未盲目套用常规的自我对弈（Self-play）范式，而是深刻剖析了生存博弈中的动态风险权重，创新性地提出了基于“人格塑造（Personality Shaping）”的奖励工程机制，成功克服了传统多智能体训练中策略同质化与极易陷入局部纳什均衡的底层难题。
  • 技术路线：本项目基于不对称的 Actor-Critic 架构构建了智能体决策网络，利用 Transformer 编码器深度挖掘对手历史行为序列中的长程博弈依赖与欺骗模式，结合多层感知机提取的当前静态局势，输出解耦的“质疑”与“出牌”双分支动作指令；在训练策略上，设计了“同质化自我对弈”与“异质化混合对弈”的两阶段课程学习机制，首先通过定制化的内在奖励函数，预训练出具备“激进欺骗”、“激进质疑”和“保守生存”等极端特征的基线智能体群体构建生态，随后将无偏置的理性智能体置于该异质化环境中进行对抗演化，迫使其在极高的死亡惩罚约束下，基于信息优势进行动态的贝叶斯策略自适应调整。
  • 项目成效：在大规模蒙特卡洛模拟测试中，演化出的理性智能体展现出远超单一特征对手的鲁棒性与决策智慧。其以 1.75 的平均生存得分及 66% 的策略成功率位居榜首，显著优于激进欺骗者（1.09）与纯保守者（1.61）。实证定性分析表明，该智能体成功学会了利用历史上下文动态推断对手的隐藏人格，并能根据左轮手枪弹巢状态（生存概率）自适应切换激进或避险策略。该成果有力证明了在极端生存博弈中，基于长程记忆与信息优势的动态风险管理策略具有压倒性的理论优越性。

基于强化学习的法医物证多人身份鉴定问题研究   项目负责人   2025.04-2025.06

法医物证鉴定在刑事侦查中具有关键作用，但传统分析方法在处理多人混合样本的STR（短串联重复序列）图谱时，面临等位基因筛选易受干扰、混合比例估计不准、联合推理缺乏统一机制等严峻挑战。本项目创新性地构建了法医物证领域的首个多智能体强化学习架构，通过多算法协同，系统性地解决了多贡献者身份识别中的核心决策难题。

  • 个人职责：作为项目负责人及论文第一作者，我主导了面向法医学复杂场景的全流程强化学习建模与底层算法开发。具体而言，我负责将高度复杂的STR图谱解析任务拆解为马尔可夫决策过程（MDP），并针对人数识别、比例估计、图谱去噪和基因型推断四大子任务，独立设计并训练了四个协同工作的强化学习智能体。同时，我主导设计了融合多项非线性特征的动态奖励函数，成功攻克了法医物证中极端样本比例不平衡与高噪声环境下的模型发散难题。
  • 技术路线：本项目采用多算法协同策略实现端到端智能分析：在人数推断任务中，使用引入分层加权奖励的 REINFORCE 算法进行离散分类，有效缓解类别不均衡；在贡献度分析中，通过 Soft Actor-Critic (SAC) 算法结合 JS 散度和均方误差（MSE），在连续动作空间下实现极高精度的混合比例回归；在信号处理环节，设计结合深度确定性策略梯度 (DDPG) 与优先级经验回放 (PER) 的智能体进行波形自适应降噪，并引入 L1/L2 正则化以最大限度保留生物学特征；最终在身份推断环节，创新性提出融合蒙特卡洛树搜索（MCTS）与束搜索（Beam Search）的两阶段多证据链联合推理模型，以策略网络高效引导高维基因型组合的最优匹配。
  • 项目成效：该多智能体框架在各项核心任务中均展现出超越传统经验规则的优异性能。其中，REINFORCE 智能体的人数识别峰值准确率达 97.56%；SAC 模型在极端比例配置（如 1:9）下的预测准确率达 92.3%，显著优于传统最大似然估计法的 78.5%；DDPG 降噪模型成功提升了 15.7% 的降噪稳定性；MCTS 联合推理机制在 2-3 人混合样本的基因型还原准确率分别高达 92.3% 和 85.1%。该研究成果为法医物证鉴定领域提供了全新的、具备极高鲁棒性的智能化技术路径。

全屋智能设备权限管理与多用户共享系统（YatSpark）   项目负责人   2025.04-2025.06

随着物联网（IoT）技术的普及，全屋智能逐渐从个体使用迈向多用户协同场景。然而，现有智能家居系统普遍面临权限控制单一、共享逻辑混乱、使用行为难以追溯等痛点。本项目提出并开发了 YatSpark 系统，旨在通过多角色身份认证、设备权限细粒度分配与严格的操作日志审计，构建一个安全、可控、防越权的多用户智能家居共享环境（如高校宿舍、酒店公寓等场景）。

  • 个人职责：作为项目负责人，我全面统筹了系统的需求分析、架构设计与前后端开发全流程。具体而言，我主导设计了“房间-模板-个人权限”三级鉴权模型与数据库实体映射方案，独立完成了核心的拦截器鉴权逻辑与设备状态冲突处理机制。此外，我还负责组织了基于测试驱动开发（TDD）的系统功能与性能验收，确保系统在高并发访问下仍能维持良好的稳定性与响应效率。
  • 技术路线：本项目采用前后端分离架构实现跨平台统一协同。前端利用跨平台框架完成一套代码多端（网页、小程序、App）适配，并建立条件编译机制保障各端交互体验的一致性；后端构建了标准化的 RESTful API 接口，并引入无状态的令牌认证机制实现安全的高并发鉴权。在核心权限设计上，系统摒弃了传统的粗放式全局控制，实现了精确到“具体设备-特定操作（如温度调节、亮度设定）”的微观颗粒度白名单授权逻辑；针对并发控制与数据一致性难题，在业务逻辑层实现了全局事务的统一接管，并设计了设备状态版本校验机制，从底层化解了多用户同时操控同一设备可能引发的指令冲突与状态覆写风险。
  • 项目成效：系统在功能与性能测试中表现出极佳的稳定性。核心模块（如权限隔离、精细授权、电器操作、分级日志审计）全部通过TDD测试用例验证。在压力测试下，系统CPU占用率稳定低于10%，内存消耗控制在50MB以内，平均响应时延不超过1秒，完全满足多用户并发控制的实际部署要求，具备高可靠的商业化上线条件。

智能家庭场景下隐私保护的视频异常行为检测算法研究   项目负责人   2023.12-2024.12

随着智能家居监控设备的普及，家庭视频数据的隐私泄露风险日益凸显。传统监控系统在云端直接处理原始视频极易引发安全隐患，而纯边缘计算又常受限于硬件算力瓶颈。本项目旨在研究并开发一套“端云协同”的新型视频异常行为检测框架，在物理阻断敏感隐私外泄的前提下，实现高效的家庭安防异常监控。

  • 个人职责：作为校级大创项目的负责人，我全面主导了整体端云协同技术路线的设计与软硬件架构的搭建。在具体攻坚中，我重点负责了云端异常行为检测算法的开发与算力部署，独立完成了基于深度学习的时空特征建模与网络训练。同时，我统筹团队成员完成了树莓派硬件端的视频流采集与边缘侧隐私模糊处理模块，确保了软硬件跨端链路的无缝集成与通讯。
  • 技术路线：本项目创新性地采取了“边缘侧隐私脱敏，云端侧异常推理”的解耦架构。在边缘侧（基于树莓派），引入 SSD (Single Shot MultiBox Detector) 目标检测框架高效锁定画面中的人形及隐私敏感区域，并利用动态高斯滤波技术进行物理级的局部模糊遮蔽，确保上传至云端的数据流被彻底脱敏；在云端服务器侧，构建了 CNN-LSTM 混合神经网络模型，通过 CNN 网络提取多帧图像的深层空间拓扑特征，利用 LSTM 网络建模视频序列之间的长程时序因果动态，最终经由全连接层分类器高维映射出行为异常（如跌倒、入侵）的概率判决。
  • 项目成效：项目成功打通了从硬件采集、边缘脱敏到云端推理的端到端闭环，构建了具备高可用性的智能安防原型系统。测试结果表明，该系统在完成异常报警任务的同时，有效化解了家庭安防与隐私泄露的底层矛盾。此外，立足于真实的工程落地标准，团队客观审视了当前系统在极高帧率下的推理时延瓶颈与缺乏精确空间定位等局限性，为该算法后续向边缘轻量化部署与商用级产品化演进沉淀了高价值的优化路径与架构经验。
